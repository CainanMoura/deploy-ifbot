# 🤖 Chatbot IFCE Acopiara

Assistente virtual para WhatsApp focado em automação de serviços acadêmicos e respostas institucionais, utilizando **Node.js** para a interface de chat e **Python (Flask + Groq AI)** para o processamento de linguagem natural.

## 🚀 Tecnologias

- **Interface (WhatsApp):** Node.js, `whatsapp-web.js`, `qrcode-terminal`.
- **Backend (IA):** Python, Flask, Groq SDK (LLaMA 3.1).
- **Integração:** REST API (Axios).

## 📋 Pré-requisitos

- Node.js (v18+)
- Python (v3.10+)
- Conta/Chave de API na [Groq](https://console.groq.com/)
